Usuário

O sistema tera uma abordagem diferente para cada tipo de usuario

Asseguradora

Acesso ao Backoffice, fornece servico de seguranca para o a transportadora

Admin

Acesso ao Backoffice, manipulacao e vizualizacao de dados

Transportadora

Realizara o tranporte da carga devendo seguir as especificacoes fornecidas

Cliente (Receptor)

Recebera o transporte realizado

Trajetos

O sistema girara em torno de trajetos manipulados pelos usuários

Viagem

Viagens englobarão rotas com suas respectivas paradas

Estabelecimento (Paradas)

Uma rota podera ter paradas especificas e programadas

Rota

Uma rota partira de dois pontos, inicio e fim podendo haver intermediacoes(paradas)

Backoffice

Disponível apenas para alguns tipos de usuários

Chatbot

Servico de AI com Python (Artificial Intelligent) para usuários e empresas

Duvidas Frequentes

Auxílio a usuários em questoes comuns com auxiliará processos

Notificações

Realizara atualizações sobre processos enviando pelo whatsapp ou telegram

Help Center
